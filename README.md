# 🎮 TicTacToe_CLI_App

A fun, terminal-based 2-player Tic-Tac-Toe game written 100% in Java. No GUI required!

## ✅ Features
- 2 Player (X vs O)
- Input validation
- Detects wins & draws
- Easy to compile and run

## 🧑‍💻 How to Run

### 1. Compile

```bash
javac -d out src/com/sam/tictactoe/TicTacToe.java
